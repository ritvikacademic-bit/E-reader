Backup & Restore

Move everything — highlights, notes, ink, reading position, stats, reading list, genres, themes and settings — from one copy of Lumen to another. Works between the file you open from WhatsApp and the one you host online, in either direction.

Leave this off unless you need it. Without it the backup is small and fast, and your library still comes back — just re-add the same PDF files afterwards and every highlight, note and page position re-attaches to them automatically. With it on, the file can be hundreds of megabytes and may run the tablet out of memory.

Voices

Three ways to read aloud. All free, all unlimited. Pick whichever your tablet handles best.

Android's own voices. Offline, instant, zero download. Quality depends on which voice pack you have installed.

Make these sound much better — on the tablet open Settings → Accessibility → Text-to-speech output, set the engine to Google Text-to-speech, then Install voice data → English (United States) and download the voices marked as high quality. Reopen Lumen afterwards.

Piper neural voices run inside this page with WebAssembly. Free, unlimited, and offline once the voice is downloaded.

Tip: on an older tablet pick a -low voice. It's ~10 MB and generates audio several times faster than a -medium one.

Run a free TTS server on your laptop (Piper HTTP, Kokoro-FastAPI, openedai-speech…) and let the tablet fetch audio from it over Wi-Fi. The laptop does the work; the tablet just plays.

If this page is on https://, Chrome will block a plain http:// server. Serve Lumen from the same machine over http and the problem disappears.
